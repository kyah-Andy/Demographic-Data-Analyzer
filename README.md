## 📌 Overview

This project analyzes demographic data from the 1994 U.S. Census using **Python and Pandas**.  
It performs statistical analysis to uncover insights about education, income, work hours, and occupation trends.

---

## 🚀 Features

- Race distribution analysis
- Average age of male population
- Education level statistics
- Income comparison by education level
- Minimum working hours analysis
- Country-based income distribution
- Occupation analysis for high earners in India

---

## 📂 Dataset

UCI Adult Income Dataset:  
https://archive.ics.uci.edu/ml/datasets/adult
